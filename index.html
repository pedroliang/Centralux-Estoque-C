Configurar sincronização na nuvem (Supabase)

Os dados passarão a ser salvos no Supabase e sincronizados em tempo real entre dispositivos.
Pegue Project URL e anon public key em: Supabase Dashboard → seu projeto → Settings → API.
SQL para criar a tabela (copie no SQL Editor do Supabase) 
-- rode uma vez no SQL Editor do seu projeto Supabase:
create table if not exists items (
  id uuid primary key default gen_random_uuid(),
  codigo text not null unique,
  qtd integer not null default 0,
  localizacao text,
  updated_at timestamptz default now()
);
alter table items enable row level security;
create policy "leitura publica"   on items for select using (true);
create policy "insercao publica"  on items for insert with check (true);
create policy "update publico"    on items for update using (true) with check (true);
create policy "delete publico"    on items for delete using (true);
alter publication supabase_realtime add table items;
